PEP-Visualize
=============
Visualize the relationship between each pair of Python Enhancedment Proposals (a.k.a. PEP)
- Keywords, such as Dictionary, Enum, etc.
- Link between PEPs

Installation
------------
- Please bower to install all javascript libraries. See bower.json
